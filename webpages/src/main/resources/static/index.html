Rest requests

Send a HTTP GET request to http://localhost:8080/users and you will receive two example users

Send a HTTP GET request to http://localhost:8080/user/{name_of_your_choice} and you will receive a user with the name you put

Send a HTTP POST request to http://localhost:8080/user with a JSON body with name and email then you will receive a user with the data you put
